Reaction Monkey

How to Play

Objective

Click on squares that match the target color as quickly as possible to score points!

Scoring

  • +10 points for clicking correct color
  • -5 points for clicking wrong color

Game Rules

  • Colors change automatically based on difficulty
  • 1-3 squares will match the target color
  • Game ends when timer reaches zero
