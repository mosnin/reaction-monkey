## Local Development

1. Clone the repository:
```bash
git clone https://github.com/your-username/reaction-monkey.git
```

2. Open the project directory:
```bash
cd reaction-monkey
```

3. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
```
